# v2.7.1｜五种模式完整任务清单

日期：2026-07-16
范围：网页版任务中心与商业任务库

## 新增

- 任务模式调整为：家庭、聚会、情侣、轻松全集、手动任务。
- 任务中心增加当前模式状态、玩家人数、存档状态和直接开始游戏按钮。
- 家庭、聚会、情侣各自拥有完整的六类任务清单：起飞、基地掷骰、外圈、直道、国王卡、终极任务。
- 轻松全集通过目录继承合并前三种模式，保留六类任务分类。
- 网页游戏根据当前模式和触发类型抽取任务。

## 规则

- 起飞只从当前模式的起飞清单抽取。
- 基地根据骰子 2、3、4、5 从对应清单抽取。
- 外圈只从外圈清单抽取。
- 直道只从直道清单抽取。
- 国王卡只在骰子为 1 时触发。
- 游戏结束时从当前模式终极任务清单抽取。
- 手动任务模式继续使用原来的任务编辑和导入导出功能。

## 验证

- 四个商业模式的六类清单均已检查为非空。
- 所有目录引用的任务 ID 均能对应到任务文案。
- 网页内嵌脚本通过 `node --check`。
- 任务库 JSON 通过 JSON 格式检查。

## 小程序同步

本轮只修改网页版，未修改 `minigame/`。小程序需要按同样的模式目录与六类抽取规则同步实现。
