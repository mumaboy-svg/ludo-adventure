# v2.15.0 小游戏当前玩家 HUD 同步

日期：2026-07-19

## 本次更新

- 当前玩家栏增加与棋子颜色一致的飞机头像。
- 显示玩家名称、当前回合状态、基地/外圈/直道/终点位置阶段。
- 显示当前轮次和上次骰子点数，沿用现有骰子 transaction 的最终结果。
- 玩家切换时提供轻量高亮反馈，低动态设置下保持静态显示。
- 竖屏、矮屏和横屏为原生胶囊、棋盘、骰子和操作区预留空间。

## 保持不变

- 自由昵称、云审核和既有存档兼容逻辑。
- 骰子唯一 transaction、点数显示、棋盘坐标、直道入口、终点反弹和任务触发。
- 网页端、C3 任务中心和小游戏骰子抛出轨迹。
