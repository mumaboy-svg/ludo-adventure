# v2.7.6 · A6 任务中心基础美化

日期：2026-07-16
更新范围：网页版 + 微信小游戏版
更新类型：视觉优化（中低风险）

## 本次修改

### 网页版

- 将任务中心主标题升级为“任务卡册”，增加卡册装订、章节标识和更清楚的任务状态层级。
- 五种任务模式继续沿用家庭、聚会、情侣、轻松全集、手动任务，增加悬停高光和“已启用”标记。
- 任务总数与任务状态改为更醒目的信息卡。
- 手动任务的读取、保存、导出、示例、检查和清空入口改为图标化工具网格。
- 手机窄屏下工具按钮自动切换为双列，避免文字与按钮拥挤。

### 微信小游戏版

- 任务中心标题改为“任务卡册”，增加装订式主面板。
- 在模式页直接显示当前启用任务数量。
- 自定义任务页增加分类图标、装订效果、任务卡边框和填写状态色条。
- 保留原模式选择、任务编辑、导入导出、任务检查和开始游戏逻辑。

## 未修改范围

- 未修改任务内容与商业任务库。
- 未修改任务触发顺序或任务抽取逻辑。
- 未修改棋盘坐标、移动、终点反弹、骰子和存档结构。
- 未增加图片素材，本轮不需要用户制作图片。
