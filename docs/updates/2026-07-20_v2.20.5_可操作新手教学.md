# 小游戏 v2.20.5 可操作新手教学

日期：2026-07-20

## 目标

“糖果云国启程信”继续负责交代糖果公主、云堡和四色飞行小勇士的故事。本次补上首局真实对局内教学，避免新玩家只看完故事就直接面对棋盘。

## 玩家流程

1. 首次完成或跳过故事后，从“新游戏”或“快速开始”进入一局新棋盘。
2. 教学遮罩只高亮真实骰子区域，玩家必须点击骰子完成第一次掷骰。
3. 原有骰子 transaction、动画、移动、回合和任务照常执行。玩家关闭首次真实任务卡后，会看到骰子结果与“掷出 6 才能起飞”的说明。
4. 玩家继续查看任务卡说明后，教学结束，进入正常救援。

首掷的六种结果都有既有真实任务卡：`1` 点为国王卡，`2-5` 点为基地任务，`6` 点为起飞任务。因此教学不会凭空生成任务或篡改骰子结果。

## 兼容与范围

- 新完成状态使用独立本地键 `ludo_teaching_guide_v1`，不写入 `ludo_minigame_state_v3`。
- 看过故事的旧用户下一次点“新游戏”或“快速开始”仍会补一次操作教学。
- 重看故事、继续上局不会触发教学；“清空全部数据”会一并清除教学完成状态。
- 低动态模式保留完整教学内容，仅停止骰子高亮的呼吸缩放。
- 未修改骰子 transaction、棋盘、移动、任务触发、排行、云函数、网页端、昵称审核和对局存档结构。
