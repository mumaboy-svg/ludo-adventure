# v2.16.0 小游戏任务中心卡册化

日期：2026-07-19

## 本次更新

- 任务中心补充当前模式、玩家数量、存档状态与牌组摘要。
- 五种任务模式保持可直接切换，当前模式继续使用明确选中状态。
- 开始入口会根据存档显示“开始游戏”或“继续上局”。
- 横屏改为五张紧凑模式卡同排；手动任务在横屏使用单条分页，底部保存按钮保持在安全区内。
- 竖屏继续保留双列卡册与逐条编辑、读取、保存、导入、导出、检查和清空数据入口。

- 骰子随机与最终点数显示契约。
- 棋盘坐标、起飞、直道、飞跃、终点、任务触发和胜利判断。
