## 修改目标

- 将设置页从基础表单升级为完整游戏菜单。
- 统一玩家配置、声音与画面、数据管理三类设置。
- 桌面端提供清晰的侧边分类导航。
- 手机端提供紧凑的顶部分类标签。
- 为 `844×390` 矮横屏建立专项布局，避免内容和底部操作互相遮挡。
- 让保存、恢复默认、继续上局、开始新游戏和危险操作获得明确反馈。

## 实现内容

- 玩家配置区集中展示玩家数量、名称、颜色、存档状态和开局覆盖提示。
- 玩家配置支持独立保存，页面重新打开后可恢复上次设置。
- 背景音乐和动态效果使用统一的游戏化开关样式。
- 数据管理区展示任务与存档状态，并区分恢复默认和清空全部数据。
- 恢复默认只重置玩家、声音和动态效果偏好，不删除游戏存档或任务数据。
- 底部操作区统一放置继续上局、保存设置和开始新游戏。
- 页面操作提供可见反馈，降低用户对保存结果的不确定感。
