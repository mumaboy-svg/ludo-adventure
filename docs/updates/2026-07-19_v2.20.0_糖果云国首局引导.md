# v2.20.0 糖果云国首局引导

## 玩家可见变化

- 首次点击“新游戏”或“快速开始”时，会先看到两页“糖果云国启程信”。
- 故事设定为：糖果公主被困在棉花糖云堡，红、黄、蓝、绿四位飞行小勇士出发救援。
- 引导说明骰子代表顺风，任务卡是勇气试炼，国王卡是云国魔法。
- 首页新增“故事”入口，可随时重看。

## 流程与兼容

- 完成或选择“稍后再说”后，会继续原本的“新游戏”设置页或“快速开始”新局流程。
- “继续上局”不会触发或被首局引导拦截。
- 引导状态使用独立本地键 `ludo_story_guide_v1`，不修改 `ludo_minigame_state_v3` 对局存档结构。
- 低动态模式保留静态可读布局，不播放四色飞机浮动效果。

## 范围

- 本轮只修改小游戏首局体验，不改网页端、骰子随机 transaction、棋盘、移动规则、任务内容/触发、昵称云审核、云函数或 Git/微信发布状态。
- 首版复用现有标题 Logo、四色飞机和首页糖果装饰；糖果公主与棉花糖云堡的专属插画由美化任务后续补入。
