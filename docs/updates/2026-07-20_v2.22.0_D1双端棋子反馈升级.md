## 具体改动

### 网页版

- 当前棋子光环改为更清晰的玩家色细环，减少对棋盘格内容的遮盖。
- 移动中的棋子增加贴地椭圆阴影，保留原有弧线、轻微倾斜与抬升。
- 飞跃和逐格移动完成后统一触发落地扩散环与压缩回弹。
- 落地状态不会延迟原完成回调，任务卡、国王卡和胜利处理仍按既有顺序执行。
- 浏览器低动态偏好和游戏内低动态设置都会关闭非必要循环与落地运动。

### 微信小游戏

- Canvas 棋盘为当前可行动棋子绘制白色内圈与玩家色外圈。
- 棋子移动时按轨迹进度绘制地面阴影，并增加轻微倾斜和缩放。
- 动画结束后以独立瞬态绘制落地扩散环和一次压缩回弹。
- 低动态模式缩短移动时间、取消移动弧线，并跳过落地动画。
- 落地瞬态不写入存档，也不改变玩家、格子或回合状态。
