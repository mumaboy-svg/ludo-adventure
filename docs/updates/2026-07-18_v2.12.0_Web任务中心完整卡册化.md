- 五种任务模式增加独立色彩、图标、说明和“使用中”状态。
- 增加任务总数、任务状态、玩家数量、存档状态和当前牌组摘要。
- 预设模式集中显示任务卡册并隐藏编辑区。
- 手动模式恢复任务编辑、导入导出和保存入口。
- 将清空数据从普通工具中分离为危险操作。
- 手机端采用双列模式卡布局，手动任务卡占整行。
- 增加矮横屏紧凑布局，压缩说明、卡片高度和开局区域。
- 进入任务中心时重置滚动位置。
