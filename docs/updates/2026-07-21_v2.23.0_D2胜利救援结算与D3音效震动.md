# v2.23.0 D2 胜利救援结算与 D3 音效震动

日期：2026-07-21

## 版本评估

- 类型：功能版本。
- 版本路径：双端正式基线 `v2.22.1` 升级为 `v2.23.0`。
- 原因：新增完整结算模块、两类结算视觉、双端短音效，以及小游戏独立震动设置，属于玩家可感知的一组新功能。

## D2 胜利与救援结算

- 普通胜利显示“飞行勇士凯旋”结算，使用金色飞机冲线主视觉。
- 首次糖果云国救援完成显示独立救援结算，使用救援徽章主视觉。
- 横屏和竖屏分别使用独立结算框素材。
- 排名快照按完赛顺序生成，兼容两人、四人和四人以上对局。
- 结算加入程序绘制的星星和彩带；低动态模式关闭持续彩带。
- 结算提供“返回大厅”和“再来一局”；再来一局清除旧结算并创建新对局。
- 结算等待终局任务、救援说明、延迟弹窗、棋子移动和掷骰结束后再打开。

## D3 音效与震动

- 双端接入棋子移动、棋子落地、任务卡翻出、国王卡揭示和胜利短音效。
- 网页版设置新增独立游戏音效开关，背景音乐开关继续独立工作。
- 小游戏设置新增游戏音效和震动反馈开关，使用独立本地键持久化。
- 小游戏在任务卡、国王卡、棋子落地和结算等关键节点使用克制震动。
- 音效播放失败、设备不支持震动或用户关闭开关时安全降级，不阻断玩法。
