# v2.11.0 Web 当前玩家 HUD

## 版本评估

- 类型：功能版本 / 完整 UI 模块。
- 版本：网页版由 `v2.10.1` 升级为 `v2.11.0`。
- 范围：网页版主游戏界面 HUD；微信小游戏暂不落盘同步。

## 修改内容

- 当前玩家区域增加对应颜色的飞机头像、玩家名称和回合状态。
- 增加当前位置或阶段提示，区分基地、外圈、终点直道和完成状态。
- 将当前轮次与上次点数整理为独立信息层级。
- 玩家切换时增加轻量入场动画，并支持减少动态效果偏好。
- 修复 `844×390` 等矮横屏下游戏区被压成单列的问题。
- 移除横版大厅三个主按钮后方的长玻璃底，减少背景遮挡。

## 未修改内容

- 未修改骰子随机、点数结果、棋盘坐标、起飞与直道规则、任务触发、存档结构或任务库。
- 未修改 `minigame/` 的当前玩家 Canvas HUD。

## 双端同步

- 网页版：已实现。
- 微信小游戏：待按 `reports/visual/v2.11.0-hud/小程序当前玩家HUD同步任务单.md` 同步。
- `v2.11.1` 仅继续处理双端竖屏间距和右上角安全区，不代表小游戏 HUD 已完成同步。
