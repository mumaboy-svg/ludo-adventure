# v2.7.5｜A5 设置页基础美化

日期：2026-07-16
范围：网页版与微信小游戏版

## 本次更新

### 网页版

- 设置页重新划分为游戏设置、声音设置、视觉设置和数据管理。
- 背景音乐按钮改成带开/关状态的滑动开关。
- 玩家配置、任务中心、继续上局与开始游戏入口保留原功能。
- 清空全部数据移入红色危险操作区，并继续使用原确认流程。
- 手机尺寸下压缩分组间距，避免按钮和文字变形。

### 微信小游戏版

- 设置页标题与提示改成游戏菜单风格。
- 背景音乐同步为 Canvas 滑动开关。
- 增加声音状态、自动屏幕适配提示和红色“清空数据”入口。
- 玩家分页、改名、换色、添加玩家、保存开局、继续游戏保持原逻辑。
