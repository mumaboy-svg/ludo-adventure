# v2.7.9 网页版游戏进度与游戏记录独立页面

日期：2026-07-16
范围：网页版 UI 信息架构调整

## 本次更新

- 将网页版游戏内原本常驻侧栏的“玩家进度”和“游戏记录”拆分为两个独立页面。
- 游戏主界面新增两个入口卡片：
  - 游戏进度：查看玩家当前位置、完成状态、当前回合与排名。
  - 游戏记录：查看掷骰、移动、任务卡、国王卡等事件日志。
- 主游戏界面减少长列表占位，棋盘、骰子和当前行动信息更集中。
- 进度页和记录页都提供“返回游戏”和“返回大厅”。

## 未修改内容

- 不修改玩法规则。
- 不修改任务内容和任务触发条件。
- 不修改棋盘坐标、起飞点、直道入口、飞跃规则。
- 不修改骰子逻辑、音效逻辑或存档结构。
- 按用户要求，本轮不修改 `minigame/` 小程序文件。

## 版本说明

- 网页版：`v2.7.9`
- 小程序：仍为 `v2.7.8`，需后续由小程序会话按同步包单独更新。

## 验证

- 已对 `web/index.html` 内嵌脚本执行 Node.js 语法检查。
- 已确认新页面入口 ID 与原玩家进度、游戏记录渲染目标 ID 唯一。
- 已确认本轮 Git 暂存范围不包含 `minigame/` 文件。

## 小程序同步提示

小程序后续可参考本次网页版结构：在游戏主画面保留“游戏进度 / 游戏记录”两个入口，将玩家状态列表与事件日志分别放到独立页面或独立 tab，保持玩法逻辑不变。
