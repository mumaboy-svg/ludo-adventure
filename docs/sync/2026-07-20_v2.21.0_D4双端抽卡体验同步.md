## 可接受的平台差异

- 网页使用 DOM、CSS 动画和浏览器合成层；小游戏使用 Canvas 绘制和定时重绘。
- 两端动画时长和光效形状按平台性能分别调整，不要求逐帧一致。
- 小游戏没有键盘 Escape，对应关闭入口为卡面关闭按钮和遮罩。

## 素材同步

- 网页继续使用 WebP 卡面。
- 小游戏继续使用资源分包内 PNG 卡面。
- 卡面构图、文字安全区和任务内容保持不变，本轮不需要新增图片。
