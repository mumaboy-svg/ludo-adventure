# 小程序当前玩家 HUD 同步任务单

【小程序同步包】

本次更新类型：视觉 / 信息层级 / 响应式

网页版已改文件：

- `web/index.html`

小程序需要同步：是，后续单独执行。

同步目标文件：

- `minigame/game.js`

同步内容：

1. 当前玩家区域显示对应颜色的飞机头像。
2. 显示玩家名称、当前回合状态和位置阶段。
3. 显示当前轮次与上次点数，保证结果与骰子画面一致。
4. 玩家切换时加入轻量反馈；低性能设备允许简化动画。
5. 竖屏布局不能遮挡棋盘、骰子或原生胶囊菜单。

关键状态：

- 基地：准备起飞。
- 外圈：显示外圈格数或已行进格数。
- 直道：显示终点直道进度。
- 完成：显示名次或已到达终点。

禁止修改：

- 骰子随机与点数逻辑。
- 棋盘坐标、入口、飞跃和终点反弹规则。
- 任务触发、任务库、自定义模式和存档结构。

验证方法：

1. 至少检查一名玩家从基地、外圈、直道到终点的 HUD 文案。
2. 连续掷骰并核对 HUD 上次点数和骰子最终画面。
3. 检查常见竖屏尺寸和原生胶囊避让。
4. 运行 `node --check minigame/game.js`。
