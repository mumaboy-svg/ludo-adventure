# 小游戏 v2.19.0 骰子按网页版动效同步

日期：2026-07-19

## 范围

- 仅更新 `minigame/game.js` 的 Canvas 骰子视觉和动效参数。
- 以网页版 `v2.15.3` 的骰子三阶段时序、动态阴影和骰点比例为参考。

## 已同步

- 接触节点统一为 `.56` 与 `.82`：首次抛出、第一次落地后的小弹跳、第二次落地后的回正。
- 40px、46px、58px 骰子使用与网页小/中/大骰子台一致的横向位移、离地高度和缩放分档。
- 骰点半径改为按骰身尺寸计算，保持 1-6 点可读性和实体三面比例。
- 普通动效为 `820ms`，低动态模式为 `180ms`。

## 未改动

- 骰子唯一 transaction、随机取值次数、最终点数先绘制后解锁和 `resolveDiceRoll(transaction.value)`。
- 棋盘、棋子移动、任务、存档、昵称审核、HUD、任务中心、按钮按压反馈和网页版。

- `node --check minigame/game.js`
- `node minigame/tests/dice_c5_test.js`
- `node minigame/tests/motion_c6_test.js`
