# v2.7.0 A4 弹窗基础统一

- 日期：2026-07-16
- 更新规模：补丁更新
- 版本归属：并入同期双端 `v2.7.0` 功能版本

> 并行会话在 A4 实施期间加入商业任务库和任务包模式，并将小程序升级到 `v2.7.0`。A4 保留为该版本的弹窗视觉子项，不再单独占用 `v2.6.4` 版本号。

## 网页版

- 任务卡与国王卡统一使用暖黑色渐变遮罩和轻度背景模糊。
- 卡片增加约 200ms 的缩放、上移和淡入反馈，关闭时使用短暂淡出。
- 在图片右上安全位置增加圆形关闭按钮，底部确认按钮继续放在原卡面按钮区域。
- 支持四种关闭方式：底部“完成，继续”、右上关闭、点击卡片外遮罩、Esc 键。
- 增加关闭阶段队列锁，连续任务不会在退场动画中重叠显示。

## 微信小游戏版

- Canvas 弹窗同步统一遮罩透明度、卡片阴影和约 190ms 的缩放淡入。
- 增加右上圆形关闭触控区，底部使用独立绘制的金色确认按钮。
- 点击卡片外遮罩、右上关闭或底部确认都会沿用原有 `closeCurrentModal()` 队列处理。
- 任务卡和国王卡继续使用现有用户素材及原文字安全区。

## 未修改

- 任务和国王卡文字内容。
- 弹窗触发顺序及任务队列数据结构。
- 骰子、棋盘坐标、移动、直道入口、飞跃和终点反弹规则。
- 游戏存档结构、A2 玩家进度、A3 游戏记录及背景音乐。
- 网页原生的危险操作确认框和小程序系统输入框；它们将在设置页专项中处理。
