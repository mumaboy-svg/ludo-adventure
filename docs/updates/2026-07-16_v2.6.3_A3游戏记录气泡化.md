# v2.6.3 A3 游戏记录气泡化

- 日期：2026-07-16
- 更新规模：补丁更新
- 版本变化：`v2.6.2 → v2.6.3`

## 网页版

- 将普通文字日志升级为事件气泡，加入图标、类型标签、时间和最新标记。
- 显示层自动识别掷骰、移动、任务、国王卡、提示、进展和系统事件。
- 最新事件使用对应类型的描边和轻量光圈突出，不使用持续动画。
- 长文本允许按任意位置换行；手机版继续使用紧凑字号和约 140px 日志区域。
- 兼容旧字符串日志、缺少 `at` 时间的旧记录和现有结构化日志。

## 微信小游戏版

- “游戏记录”同步重绘为浅色事件卡片，左侧显示类型图标，顶部显示类型标签与时间。
- 当前分页中的最新事件使用类型色描边并标记“最新”。
- 单条记录最多显示两行，卡片和分页数量按可用高度计算，避免与翻页按钮重叠。
- 事件类型沿用网页版的文字推断规则，不改变已有日志数据结构。
