# 网页版 v2.20.5 糖果云国故事教学与救援结算

日期：2026-07-20

## 版本评估

网页版从 `v2.15.3` 更新到 `v2.20.5`。本轮补齐与已发布小游戏 `v2.20.5` 一致的玩家可见开局体验：两页故事、首局真实操作教学和首次救援结算；不改变棋盘、随机或既有对局存档契约。

## 玩家可见改动

- 新游戏与快速开始在首次进入时展示“云堡的求救信”和“四色飞行小勇士”两页故事。
- 故事完成后的首局会高亮真实骰子；玩家必须点击真实骰子，原有随机 transaction、移动和任务照常执行。
- 首次真实任务卡关闭后，依次解释刚才的骰子结果和“掷出 6 才能起飞”，再说明任务卡用途；完成后进入自由救援。
- 大厅新增“糖果云国故事”入口，只重看故事，不再次触发教学；继续上局不触发教学。
- 首次完整对局结束后，在既有终局任务之后追加“糖果公主获救！”提示，并授予一次“糖果云国救援徽章”。

## 存储与清空数据

- 故事、教学和救援徽章使用独立本地键：
  - `flight_adventure_story_guide_v1`
  - `flight_adventure_teaching_guide_v1`
  - `flight_adventure_rescue_badge_v1`
- 不写入既有网页对局存档 `flight_adventure_v66_state`。
- 清空全部数据会清除教学与救援徽章，按小游戏已确认的产品契约保留故事完成状态。

- 微信小游戏源码、微信审核、微信正式发布和 Git tag。
