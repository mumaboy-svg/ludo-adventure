飞行棋大冒险 - 骰子音效修改说明
======================================

修改时间: 2026-06-09

修改内容:
----------
1. 将 playDiceRollSound() 函数中的白噪音音效替换为真实的骰子滚动音效
2. 使用 dice_roll.mp3 文件作为骰子滚动音效
3. 添加了降级方案：当音频文件无法播放时，自动降级到原来的白噪音音效
4. 保持 playDiceResultSound() 函数不变（显示点数时的音调音效）

技术实现:
----------
- 使用 HTML5 Audio API 播放 MP3 文件
- 音频文件路径: ./assets/audio/dice_roll.mp3 (相对于 index.html)
- 音量设置: 0.5 (50%)
- 预加载机制: 首次调用时创建Audio对象并预加载
- 重置播放: 每次播放前重置 currentTime = 0，支持快速连续点击

文件清单:
----------
✓ index.html (已修改)
✓ assets/audio/dice_roll.mp3 (音效文件)

测试方法:
----------
1. 用浏览器打开 "index.html"
2. 点击骰子或"掷骰子"按钮
3. 应该听到真实的骰子滚动声音（而不是原来的白噪音）
4. 骰子停止后，仍会播放对应点数的音调音效

注意事项:
----------
- 确保 dice_roll.mp3 文件位于 assets/audio/ 目录下
- 某些浏览器可能需要用户交互后才能播放音频（点击页面任意位置）
- 如果音频播放失败，会自动降级到白噪音音效（控制台会输出错误信息）

回滚方法:
----------
如需恢复原来的白噪音音效，将 playDiceRollSound() 函数恢复为原来的实现即可。
