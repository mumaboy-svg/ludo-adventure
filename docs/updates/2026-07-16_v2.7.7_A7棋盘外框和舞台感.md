# v2.7.7 A7 棋盘外框和舞台感

发布日期：2026-07-16
更新级别：补丁版本
更新范围：网页版与微信小游戏版

## 本次修改

- 网页版棋盘增加暖木桌游底座、双层描边、底部厚度和落地阴影，让棋盘与页面背景拉开层次。
- 网页版棋盘上沿增加当前玩家颜色光带；玩家切换或游戏结束时会同步更新颜色。
- 微信小游戏同步木质底座、内框高光、底部厚度、阴影和当前玩家颜色光带。
- 手机尺寸使用更紧凑的边框与阴影参数，避免挤压棋盘第一屏布局。
- 修复网页版进入游戏时没有启用 `.game.active`，导致游戏舞台与手机版布局样式未完整生效的问题。

## 未修改内容

- 未改变棋盘图片的绘制范围。
- 未改变外圈、直道、基地或棋子坐标。
- 未改变起飞、移动、终点反弹和胜利判断规则。
- 未改变骰子、任务触发、音效和存档结构。
- 未增加图片素材。
