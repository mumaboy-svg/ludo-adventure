# v2.9.0 B 阶段大厅与卡面素材升级

**完成日期**：2026-07-17
**更新类型**：双端视觉素材升级
**版本范围**：网页版、微信小游戏版

## 本次更新

- 横版大厅替换为新的 `1920×1080` 冒险桌游背景。
- 手机版及微信小游戏大厅替换为新的 `1080×1920` 竖版背景。
- “继续上局 / 新游戏 / 快速开始”三张主按钮统一为 `900×300`、3:1 比例；网页版直接使用透明 WebP，小游戏使用透明 PNG 并在 Canvas 中绘制按钮底图。
- 任务中心、设置、棋子测试图标统一为 `384×384`，并保留棋子测试放大镜原有半透明玻璃效果。
- 新增 `384×384` 导入导出图标，网页版任务中心的读取、导出入口开始使用该素材。
- 任务卡和国王卡弹窗背景替换为新的 `1000×1000` 卡面。
- 双端可见版本号统一升级为 `v2.9.0`。

## 图片处理

- 用户原始图片未被覆盖。
- 背景按目标比例中心轻裁并使用 Lanczos 高质量缩放。
- 透明按钮和图标按 Alpha 主体边界归一化、居中，缩放时保护透明边缘。
- 网页端采用 WebP；微信小游戏端采用 JPEG 背景和 PNG 透明素材，降低 Canvas 解码兼容风险。

## 保持不变

- 不修改棋盘坐标、起飞点、直道入口、特殊飞跃和终点反弹规则。
- 不修改骰子概率、骰子点数显示、任务触发、国王卡触发或存档结构。
- 不修改玩家数量与任务内容。

## 验证

- 网页版内嵌脚本和 `minigame/game.js` 通过 Node.js 语法检查。
- 21 个新增或替换后的正式图片均可正常解码，尺寸、格式和透明通道符合双端目标。
- 网页版在 `1366×768`、`390×844`、`375×667` 下完成首页渲染检查，页面无横向溢出，控制台无运行错误。
- 三张主按钮按准确 3:1 比例显示，手机版文字仍由独立文字层居中，不随背景图片拉伸。
