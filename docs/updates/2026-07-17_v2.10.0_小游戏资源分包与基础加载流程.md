# v2.10.0 资源分包、双端加载页与横屏大厅动态

## 更新范围

- 微信小游戏资源结构、启动加载流程与正式加载页视觉。
- 网页版正式启动加载页与横屏大厅轻量动态。
- 双端可见版本号同步为 `v2.10.0`。

## 主要变化

1. 新增 `packages/game-assets/` 普通分包。
2. 大厅首次显示素材保留在主包。
3. 棋盘、四种棋子、任务卡、国王卡、任务/设置背景、HUD、骰子面板和三段音频迁入资源分包。
4. 启动时先调用 `wx.loadSubpackage()`，再预载运行时图片。
5. 双端加载画面使用竖横屏专用背景、品牌 Logo、飞机跟随、四色进度条和版本号。
6. 加载进度由真实关键资源完成数驱动；关键资源失败后可点击重试，非关键装饰资源失败不阻塞进入大厅。
7. 网页横屏大厅增加轻量飞机、星光、糖果粒子和按钮入场动效，并支持低动态偏好降级。

## 包体目标

- 主包本地文件估算：约 2.15 MB。
- `gameAssets` 分包本地文件估算：约 3.19 MB。
- 微信限制为主包不超过 4 MB、全部主包加分包不超过 30 MB。

## 验证

- `minigame/game.js` 与网页内联脚本通过 Node.js 语法检查。
- `minigame/game.json` 通过 JSON 解析，运行时图片和音频引用均能在主包或资源分包中定位。
- 微信开发者工具 CLI preview 通过，主包约 2.33 MB，资源分包约 2.04 MB。
- 网页版在 `1366×768`、`1440×900`、`844×390`、`390×844` 和 `375×667` 完成加载、失败重试和无横向溢出检查。

## 保持不变

- 52 点外圈、各色起飞点和直道入口。
- 直道精确到达与超点反弹。
- 特殊飞行、任务/国王卡、存档和背景音乐开关逻辑。
- 网页版按钮 ID、页面跳转、玩法、任务和存档逻辑。

## 发布状态

- 双端源码、资源分包、正式加载页素材和版本文档已完成本地验收。
- 微信真机最终截图和正式小游戏上传仍需在微信发布流程中完成。
