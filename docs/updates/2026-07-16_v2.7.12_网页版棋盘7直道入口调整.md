# v2.7.12 网页版棋盘7直道入口调整

日期：2026-07-16
范围：网页版直道入口映射

## 新入口

- 红色：外圈第 `20` 点进入 `R1-R6`。
- 绿色：外圈第 `33` 点进入 `G1-G6`。
- 黄色：外圈第 `46` 点进入 `Y1-Y6`。
- 蓝色：外圈第 `7` 点进入 `B1-B6`。

## 实现说明

界面编号从 1 开始，代码数组索引从 0 开始，因此 `ENTRY_INDEX` 使用：

```js
{ R: 19, Y: 45, B: 6, G: 32 }
```

棋子顺时针移动到自己的入口后立即进入对应颜色直道；剩余骰子点数继续用于直道移动。
