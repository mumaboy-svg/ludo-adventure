# v2.22.1 D1 棋子反馈与基地任务修复

日期：2026-07-20

## 版本评估

- 类型：功能版本后的修订发布。
- 版本路径：远端正式版本 `v2.21.0`；D1 本地开发版本 `v2.22.0`；本轮加入基地任务缺陷修复后，双端正式发布为 `v2.22.1`。
- 原因：D1 是玩家可见的双端棋子反馈升级，基地任务变更属于规则实现修复；采用本轮最高版本并增加修订号。

## 发布内容

### D1 双端棋子反馈

- 当前可行动棋子增加玩家色光环。
- 棋子移动增加贴地阴影、轻微抬升、倾斜和缩放。
- 落地增加短促扩散环与压缩回弹。
- 低动态模式保留静态当前态并移除非必要运动。

### 微信小游戏基地任务修复

- `triggerBaseRollTask()` 不再使用 `state.triggered` 永久去重。
- 基地棋子每次掷出 `2、3、4、5` 都会触发对应任务。
- 同一架棋子重复掷出相同点数也会再次触发。
- 旧存档中的 `triggered` 字段继续保留，避免扩大存档结构变更，但不再影响基地任务。
- 延迟弹窗仍进入原 `modalQueue`，不改变任务显示顺序。
