# v2.9.1 手机版大厅背景与轻量动效

- 日期：2026-07-17
- 类型：双端手机版大厅视觉升级
- 版本：Web `v2.9.1` / 微信小游戏 `v2.9.1`

## 更新内容

1. 手机版大厅更换为云海悬浮棋盘背景，主要棋盘位于 Logo 下方，主按钮移动到下方云海区域。
2. 三个主按钮缩小，移除按钮组外层大玻璃框；左侧图标放大，“快速开始”图标额外做视觉补偿。
3. 新增小飞机每约 9 秒飞越一次。
4. 使用 CSS／Canvas 绘制 5 处低频星光和 3 颗糖果装饰，不引入整屏 GIF。
5. 三个主按钮依次弹入，稳定后仅保持约 2.5% 的轻微呼吸反馈。
6. 修复竖向宽窗口错误使用横版大厅的问题：窄屏或竖屏方向均采用手机版大厅。
7. 网页和微信小游戏版本号同步更新为 `v2.9.1`。

## 性能策略

- 背景为静态压缩图片。
- 飞机、星星、糖果和按钮动画只在大厅运行。
- 网页开启“减少动态效果”时关闭装饰与按钮动画。
- 不使用 GIF、视频或大规模粒子系统。
