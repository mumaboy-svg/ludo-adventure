## 具体改动

### 网页版

- 任务卡和国王卡使用 `rotateY` 翻出、轻抬和落定动画。
- 国王卡增加金色旋转光芒、脉冲外发光和卡面扫光。
- 点击“完成，继续”后显示绿色金边“✓ 挑战完成”章，再关闭卡片。
- 普通通知按钮改为“知道了”；关闭按钮、遮罩和 Escape 只执行关闭。
- `reduced-motion` 模式停用持续旋转和扫光，并保留静态卡面及完成章。

### 微信小游戏

- Canvas 任务卡加入翻面缩放、抬升和轻微倾斜。
- 国王卡增加旋转金色光线和环绕闪星，并由既有动画循环持续重绘。
- 任务卡和国王卡确认后绘制“✓ 挑战完成”，完成期间移除可重复点击按钮。
- 普通通知使用“知道了”并直接关闭；关闭按钮与遮罩不进入完成动作。
- 低动态模式直接显示稳定卡面，并把完成反馈缩短到 180ms。
