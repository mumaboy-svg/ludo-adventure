# v2.8.0 双端游戏进度与游戏记录独立页面

日期：2026-07-16  
更新类型：双端界面结构优化  
版本：v2.8.0

## 本次完成

- 网页版保留 v2.7.9 已完成的“游戏进度”和“游戏记录”两个独立页面，并将可见版本升级为 v2.8.0。
- 微信小游戏把原先共用一个 `records` 场景、依靠标签切换的结构拆成两个独立场景：
  - `progress`：游戏进度；
  - `records`：游戏记录。
- 微信小游戏游戏主界面新增“游戏进度”和“游戏记录”两个独立入口，与“任务中心”并列显示。
- 两个独立页面都保留“返回游戏”和“返回大厅”操作。
- 游戏进度继续显示玩家位置、当前回合、完成状态、排名、进度条和分页。
- 游戏记录继续显示掷骰、移动、任务卡、国王卡等分类事件和分页。
