# v2.17.0 小游戏实体骰子动效

日期：2026-07-19
类型：优化 / 动效

## 版本评估

本轮新增玩家可见的骰子抛出、两段弹跳、动态阴影与落地反馈，未改变骰子结果、棋盘或玩法流程，因此由 `v2.16.0` 升级为 `v2.17.0`。

## 修改文件

- `minigame/game.js`
- `minigame/tests/dice_c5_test.js`

## 改动

- Canvas 实体骰子加入横向抛出、第一次落地、第二次小弹跳与最终稳定落定。
- 阴影随骰子高度、横向位移和碰撞强度改变大小、透明度与模糊程度。
- 两次落地使用轻量圆环反馈，不增加图片或音效依赖。
- 低动态模式保留快速、可见但幅度更小的轨迹。
- 骰子专项测试覆盖两段落地、低动态、最终归零、唯一 transaction、2/5 点与连点锁。

## 保持不变

- 每次掷骰只生成一次最终随机结果。
- HUD 上次点数、最终骰面、任务回调、棋子移动与存档均使用同一最终点数。
- 昵称云审核、C2 当前玩家 HUD、C3 任务中心、棋盘坐标、直道入口、终点反弹、任务内容与存档结构未修改。
- 未修改 `web/`、云函数、Git 暂存/提交/推送或微信上传、审核、发布。
