# 发布检查表

## 通用检查

- [ ] 已确认本次发布方向：网页版 / 微信小游戏版 / 两者都发。

- [ ] 网页版入口为 `web/index.html`。

- [ ] 网页版测试页放在 `web/tests/`。

- [ ] 微信开发者工具导入目录为 `minigame/`。

- [ ] 小游戏素材路径 `assets/minigame/...` 和 `assets/audio/...` 能正常加载。
